# c-conditions-basics
Simple C programs demonstrating basic conditional statements including if, if-else, else-if, and nested if for beginners.
# C Conditions Basics

This repository contains a simple C program demonstrating basic conditional statements:
- if
- if-else
- else-if
- nested if

--- HOW TO RUN ---

1. Install a C compiler (GCC recommended):
   - Windows: Install MinGW or use an online compiler.
   - Linux/macOS: GCC is usually pre-installed.

2. Open terminal / command prompt.

3. Navigate to the project folder:
   cd c-conditions-basics

4. Compile the program:
   gcc main.c -o main

5. Run the program:
   ./main
